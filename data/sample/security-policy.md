# Security Policy

All employees must use multi-factor authentication for company systems. Passwords must not be shared through chat, email, tickets, or documents.

Customer data must be stored only in approved systems. Access to customer data requires a documented business reason and manager approval.

Security exceptions must include an owner, expiration date, compensating controls, and approval from the Security team.
