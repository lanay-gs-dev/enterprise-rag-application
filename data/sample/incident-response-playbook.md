# Incident Response Playbook

Security incidents involving customer data must be escalated immediately to the
Security team. The incident owner must document scope, affected systems,
containment actions, and customer impact assessment.

Only approved incident responders may access restricted investigation notes,
forensic evidence, or customer-impact analysis.
